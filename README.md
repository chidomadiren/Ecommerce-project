# 👗 Chichie Fashion Store (E-Commerce App)

Chichie Fashion Store is an eCommerce web application i created using Django.  The app is designed to sell dresses online and the platform provides a simple and user friendly shopping experience where customers can browse products, add items to a cart, and complete their orders through a form.


 Features

.Home page displaying store introduction and featured products  
.Product page showing available dresses with images, descriptions, and prices  
. Shopping cart system to add and review selected items  
. Order form to collect customer details (name, address, etc.)  
. Admin panel for managing products and orders  


 How to Use the App

 Home Page
When you open the website, you land on the home page. It shows an overview of the fashion store, featured items, and navigation links to other sections of the website.

 Product Page
The product page displays all available dresses. Each product includes an image, description, and price. Customers can click “Add to Cart” to select items they want to purchase.

 Cart Page
The cart page shows all selected products. Users can review their items before proceeding to checkout. This page helps users confirm their order before submission.

 Form Page (Checkout)
After reviewing the cart, users are directed to a form where they enter their personal details such as name, address, and contact information to complete their order.


Tech Stack

- Python (Django)
- HTML, CSS
- SQLite (Database)
- Bootstrap (optional if used)


 Purpose of the Project

This project was built to practice full-stack web development using Django. It helped me understand how to structure a web application, manage databases, handle user input, and build a complete eCommerce workflow from product display to checkout.
